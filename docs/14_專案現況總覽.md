# 專案現況總覽

>
> 最後更新：2026-07-24
>
> 目前版本：V3.3.1（尚未發布）

---

## 已完成功能

### 旅程與導覽

- Trip 建立、編輯、刪除與可編輯者管理。
- 每日行程依活動時間由早到晚顯示；未填時間的活動置底。
- 開啟旅程時，依出發日期自動選取對應 Day；旅程外期間預設 Day 1。
- 已登入使用者可將任一工具設為個人預設首頁；GUEST 不顯示此功能。
- 側欄工具圖示使用海水藍色，並提供預設首頁星號說明。

### 檢查清單

- 共同檢查清單：管理者可新增、編輯、刪除、勾選與排序，並同步雲端。
- USER 的共同檢查清單可新增、編輯、勾選與排序，但只保存在該帳號的本機設備。
- 私人確認清單以 `userEmail + tripId` 隔離；已勾選項目自動置底。
- 可排序卡片（共同清單、私人清單、領隊／導遊聯絡資訊、其他資訊）支援拖拉與上下箭頭；排序先更新畫面，再延後同步。
- 私人與共同清單的 USER 模式不顯示雲端複製與同步提示。

### 參考資訊與旅行工具

- 領隊／導遊聯絡資訊與其他資訊可瀏覽、管理、排序並雲端同步。
- 旅費記帳本支援多人帳本、附件、日期、分攤與 Excel 匯出。
- 外幣換算支援換匯紀錄、加權平均、臺銀參考匯率與本機離線參考資料。
- 版本資訊可由所有角色查看；目前版本不會重複出現在版本歷史。

### 權限與資料

- 角色為 GUEST、USER、trip_editor、super_admin；完整矩陣見 [權限設定資料表](13_權限設定資料表.md)。
- Trip、共同清單、私人清單、其他資訊與帳本已具 Supabase 雲端資料與權限控制；本機資料作為相容與離線備援。

## 後續待開發或改善

- 初始 JavaScript bundle 超過 500 kB；評估以動態載入降低首次開啟負擔。
- 評估 Supabase advisor 對既有 functions、RLS initplan 與 index 的建議，確認是否需要資料庫調整。
- 改善帳本附件管理體驗。
- 若多人同時修改同一份共同清單或其他資訊，目前以最後成功同步的資料為準；尚未提供衝突合併。
- 暫不規劃新的 Travel Tool（預約、行李、旅行日誌、保險、AI 助手），除非重新立案。
